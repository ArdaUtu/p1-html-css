Wat is HTML?

Het staat voor: Hyper, tekst, markup, language en dat is een: programma taal voor het schrijven van websites. Je kan het zien als een skelet van een website.

Wat is CSS?

Het staat voor cascading, style, sheets en dat betekent dat je de: vormgeving van een HTML - element vertoont op een website. Je kan het zien als een huid dus iets die de vorm geeft van een website.

Wat zijn html tags?

Het instructeerd webbrowsers over de webpagina zoals waar afbeeldingen, teksten er moeten zijn of hoe de structuur van de webpagina moet eruit zien

Wat zijn html elementen?

Het zijn zichtbare onderdelen van je website zoals tekst, afbeeldingen.

Waarom gebruiken we HTML&CSS?

Om de vormgeving en opbouw van de website, formulier,etc te gaan maken.

Waar is HTML relevant?

HTML word 96% gebruikt in websites en in websites die u regelmatig gebruikt.
